Q1 Different between Map,Filter,For Each,find

  • Map: This function is used to create a new array by applying a transformation function to each element of an existing array. The map function returns a new array of the same length as the original array, with each element transformed based on the function provided. The original array remains unchanged.
    const numbers = [1, 2, 3, 4, 5];
    const doubledNumbers = numbers.map(number => number * 2);
    console.log(doubledNumbers); // Output: [2, 4, 6, 8, 10]
  • ForEach: This function is used to loop through an array and perform a specified action for each element. It does not return anything and does not modify the original array.
    const numbers = [1, 2, 3, 4, 5];
    numbers.forEach(number => console.log(number)); // Output: 1 2 3 4 5
  • Filter: This function is used to create a new array by filtering out elements that do not meet a certain criteria. The filter function returns a new array containing only the elements that meet the specified condition.
    const numbers = [1, 2, 3, 4, 5];
    const evenNumbers = numbers.filter(number => number % 2 === 0);
    console.log(evenNumbers); // Output: [2, 4]
  • Find: This function is used to find the first element in an array that meets a certain condition. The find function returns the first element that satisfies the condition, or undefined if no such element is found.
    const numbers = [1, 2, 3, 4, 5];
    const firstEvenNumber = numbers.find(number => number % 2 === 0);
    console.log(firstEvenNumber); // Output: 2

    • Q2 Different Bewtween Local Storage,Session Storage and Cookies

  • LocalStorage: It is a web storage API that allows you to store data in the browser with no expiration date. Data stored in localStorage is available even after the browser is closed and reopened. This data can only be accessed by the domain that created it.
  • SessionStorage: It is a web storage API that allows you to store data in the browser for the duration of the session. Data stored in sessionStorage is available only during the current session and is lost when the browser is closed. This data can only be accessed by the domain that created it.
  • Cookies: Cookies are small text files that are stored on a user's computer by a website. They can be used to store data such as user preferences, login status, and shopping cart contents. Cookies can be set with an expiration date or can be set to expire when the user closes the browser. Cookies can be accessed by any page on the same domain that created them.

    • Q3 What is Event Loop and it's Working Proccess

    The event loop is a key concept in JavaScript that is responsible for handling asynchronous events and maintaining the concurrency model of JavaScript. It is a continuous loop that runs in the background of a JavaScript runtime environment, such as a web browser or Node.js, and continuously checks the event queue for new events to process.
    When an asynchronous event occurs, such as a user click, a network request, or a timer expiring, it is added to the event queue. The event loop continuously checks the event queue and when an event is found, it is removed from the queue and processed by the event loop.
    The event loop works by following these basic steps:

  • Execute any synchronous tasks in the call stack until it is empty.
  • Check the event queue for any pending events.
  • If an event is found in the event queue, remove it from the queue and execute its associated callback function (i.e., the function that was registered to handle that event).
  • If the event queue is empty, wait for new events to arrive.
    • This process allows JavaScript to handle asynchronous events in a non-blocking and efficient manner, while also ensuring that the synchronous code executes first before processing any pending events.

    Q5 What is the purpose of chooseQuiz Function ?

    This chooseQuiz function that takes two parameters: index and givenAns. The purpose of the function is to select or choose a quiz by its index and store the given answer in the answers array. The function first checks if an answer for the selected quiz already exists in the answers array using the Array.find() method. If an answer exists, it updates the givenAns property of the corresponding quiz object using the Array.splice() method.
    If the answer does not exist, the function creates a new quiz object with the givenAns property and pushes it to the answers array using the Array.push() method. The function then calls another function named displayAnswers and passes the answers array as an argument to update the display of the answer summary.
